Opo 2. Realizar a remoo da Linha que possui os dados nulos

Esta opo j pode ser utilizada para o nosso caso. A perda das linhas podem no gerar perdas significativas dependendo da futura aplicao, visto que, para alguns recursos de inteligncia artificial, tais dados podem significar a identificao de novos padres ou auxiliar na mtrica do algoritmo.

De forma geral, a remoo de dados de um DataFrame quase sempre no aconselhada pois podemos perder informaes valiosas com a excluso.

Dado a ressalva, para realizar a remoo de uma linha que possui dados faltantes, utilizaremos, junto com o comando pandas.DataFrame.drop, uma busca por valores nulos em cada uma das colunas, conforme exemplo abaixo.

In[]:
# Realiza uma cpia do DataFrame inicial para no haver alteraes nele
df_aux = df.copy(deep=True)

# Remove as linhas (index) que possuem valores nulos (NaN) na coluna 'Cidade' e salva as alteraes no prprio DataFrame
df_aux.drop(df_aux.loc[pd.isnull(df_aux['Cidade'])].index, inplace=True)

# Remove as linhas (index) que possuem valores nulos (NaN) na coluna 'Cidade' e salva as alteraes no prprio DataFrame

Opo 3. Substituir os valores faltantes pela mdia de valores da Coluna

Para os casos de variveis do tipo numricas, esta abordagem torna-se bem simples, bastando realizar a mdia numrica de acordo com os dados da coluna e substituir os valores faltantes (NaN) pela mdia.

J para dados Categricos, isso passa a ser uma abordagem complexa. Para essa base de dados, invivel realizar tal substituio, visto que os atributos que esto com falta de dados, so do tipo categricos e so relevantes para a consistncia dos dados.
